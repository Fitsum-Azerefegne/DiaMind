DiaMind – Diabetes Mental Health AI Companion 🌟
Overview
DiaMind is a web-based AI chatbot designed to support the emotional and mental health needs of individuals living with diabetes. By leveraging advanced technologies like Python (Flask) and Google Gemini AI, DiaMind delivers empathetic, evidence-based guidance, stress management strategies, and motivational support through natural conversations. 🤖💬

Features
Empathetic Conversations: Engage in real-time chat with an AI that understands the unique challenges of living with diabetes. 💖
Evidence-Based Guidance: Receive recommendations and insights grounded in scientific research and best practices. 📚
Stress Management Strategies: Access techniques and tools to help manage stress and anxiety related to diabetes. 🧘‍♀️
Motivational Support: Get encouragement and motivation to help you stay on track with your health goals. 🌈
Session History: Review past conversations to track your journey and revisit helpful advice. 📖
User-Friendly Interface: Enjoy a modern, responsive design that is accessible and easy to navigate. 🖥️
Technology Stack
Backend: Python with Flask for a robust and scalable server-side application. 🐍
AI: Google Gemini AI for natural language processing and conversational capabilities. 🌐
Frontend: Modern web technologies ensuring a responsive and engaging user experience. 🎨
Getting Started 🚀
Prerequisites
Python 3.x
Flask
Google Gemini API access
Installation
Clone the repository:

bash

git clone https://github.com/yourusername/diamind.git
cd diamind
Install the required packages:

bash

pip install -r requirements.txt
Set up your Google Gemini API key in the environment variables.

Run the application:

bash

python app.py
Open your web browser and navigate to http://localhost:5000. 🌍

Usage 🛠️
Start a conversation by typing in the chat interface. 💬
Explore various topics related to diabetes management and mental health. 🧠
Review your session history for past interactions. 📅
